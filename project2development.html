Task 1 - Multiplayer

Development

Start of the multiplayer function. At the start of the code is where I global variables so they can be used in the function. The while True loop runs a piece of code that receives an input of which user wants to start. If the input was equal to player1's name, they will start as the turns will be minus 1, initiating their turn. The opposite happens for player2. If the input is not equal to player1 or player2's name, it will print a message saying it's invalid, then repeating the question.

This part includes a loop of player1s turn, it'll ask them where they want their piece, then check if it's an invalid input or if the place has already been used. It will then insert their "O" piece onto the given part of the board.

Same code as the one above, just for player2.

Task 2 - Singleplayer

Development

Globals the variables so they can be used. It then gets the input of who starts, either the human player or the computer. If the player says yes for them to start, they will, if they say no, the computer will start. If the input does not equal to either of those it will tell them and repeat the question until there is a valid input.

Human players part of the singleplayer function, where they make their moves. It is the same code as the 2 main pieces of the multiplayer function.

The computers part of the singleplayer function. It makes a random choice from the numbers, and then checks if it's valid (hasn't been used), if it has, it will repeat until it hasn't. It then does the same as the singleplayer function but waits a couple seconds before printing so it is not immediate.
